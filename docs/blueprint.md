# **App Name**: Synapse Study

## Core Features:

- Smart Image to Text: Upload textbook chapter images. Use an AI tool to perform text extraction and analysis, so the user can edit and refine the output.
- AI Flashcard Generator: AI-powered flashcard generation that transforms key chapter topics into flashcards, creating definitions, multiple choice questions, and core concepts.
- Customizable UI: Users can edit fonts, use icons, add stickers, change and apply text formatting to notes.
- Offline Support: Offline PWA support to continue studying and making notes without an internet connection.
- Password Protection: Password protect any individual note you like.
- PDF Export: Ability to export and save notes as downloadable PDF files.

## Style Guidelines:

- Primary color: Deep Purple (#2D1B69) to set a focused, studious mood.
- Accent color: Bright Purple Glow (#C77DFF), a vibrant color to highlight interactive elements.
- Background color: Dark Navy (#0F0F23), a muted shade of the primary for comfortable contrast and a 'dark mode' feel.
- Headings: 'Poppins' (sans-serif), bold and modern for clear titles.
- Body Text: 'PT Sans' (sans-serif), a readable, clean font to enhance clarity.
- Neon glow effects on buttons and interactive elements for a modern look.
- Use subtle transitions, and card-based layouts to enhance visual appeal.
